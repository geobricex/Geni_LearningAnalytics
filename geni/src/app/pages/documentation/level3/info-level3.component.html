Level 3 represents the most comprehensive version of the academic and contextual data structure of the student. It encompasses performance indicators, interaction metrics, sociodemographic variables, and socioeconomic conditions, allowing integral, intersectional, and predictive analysis in educational contexts.

Level 3 contains a list of objects where each object represents a student with personal, academic, digital, and family information relevant for educational analysis.
  • student_id (str): Unique identifier of the student.
  • student_name (str): Full name.
  • age (int): Current age of the student.
  • gender (str): Self-perceived gender.
  • has_internet_access (bool): Indicates if the student has internet access.
  • device_type (str): Type of device used for studying.
  • previous_school_type (str): Previous school type (public/private).
  • parents_education_level (str): Educational level of parents.
  • receives_scholarship (bool): Indicates if the student receives a scholarship.
  • socioeconomic_level (str): Perceived socioeconomic level (high, medium, low).
  • family_income_monthly (float): Monthly family income in dollars.
  • dependents_count (int): Number of dependent persons in the household.
  • employment_status (str): Employment status of the student.
  • housing_type (str): Type of housing (owned, rented, irregular).
  • zone_type (str): Geographical zone (urban/rural).
  • birth_date (str): Birth date (YYYY-MM-DD).
  • civil_status (str): Marital status.
  • ethnicity (str): Self-identified ethnicity.
  • has_children (bool): Indicates if the student has children.
  • number_of_children (int): Number of children.
  • housing_ownership_status (str): Housing tenure status (owned, rented, etc.).
  • housing_condition (str): Housing condition.
  • people_living_in_home (int): Number of people living in the household.
  • rooms_in_home (int): Total number of rooms in the house.
  • commute_time_minutes (int): Daily commute time to the institution (in minutes).
  • course_id (str): Enrolled course code.
  • course_name (str): Course name.
  • accesses_last_7_days (int): Platform accesses in the last week.
  • total_connection_time_minutes (int): Total connection time in minutes.
  • resources_visited (int): Number of visited resources.
  • forum_posts (int): Number of forum participations.
  • activities (list): List of evaluated activities.
  • activity_id (str): Unique identifier of the activity.
  • type (str): Type of activity (quiz, forum, task).
  • title (str): Descriptive title of the activity.
  • grade (float): Grade obtained.
  • max_grade (float): Maximum possible grade.
  • submission_date (str, ISO 8601): Submission date and time.
  • observation (str): Teacher's comment on performance.
